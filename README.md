* install plugin  for vim

```lua
    remplasar para el comando de instalacion
```

* install Pecker for lua
```lua

```

* latex to pdf
```shell
    #Arch linux
    sudo pacman -S textlive-most
```
* Marckdown to pdf
```shell
    #Arch linux
    sudo pacman -S pandoc
```

* View pdf
```shell
    #Arch linux
    sudo pacman -S zathura
```
